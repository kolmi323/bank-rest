<h1> Система Управления Банковскими Картами (Bank Card Management System)</h1>
<p>REST API приложение для управления банковскими счетами, картами и транзакциями. Реализована ролевая модель доступа (Admin/User), шифрование данных и автоматизированное развертывание.</p>

<h3>Инструкция по запуску (Docker)</h3>

<ol>
    <li>
        Клонирование
        <p><code>git clone https://github.com/kolmi323/bank-rest</code></p>
    </li>
    <li>
        Запуск
        <p><code>docker-compose up --build</code></p>
    </li> 
    <li>
        Ожидание <b>. . .</b>:
    </li>
    <li>
        Всё готово!
    </li>
</ol>
