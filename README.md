<h1> Система Управления Банковскими Картами</h1>
<p>REST API приложение для управления банковскими счетами, картами и транзакциями. 
Реализована ролевая модель доступа (Admin/User), шифрование данных и автоматизированное развертывание.</p>

<h3>Инструкция по запуску (Docker)</h3>

    <li>
        Клонирование
        <p><code>git clone https://github.com/kolmi323/bank-rest.git</code></p>
    </li>
    <li>
